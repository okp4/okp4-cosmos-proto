## Purpose

Provides Rust [gRPC clients](https://docs.rs/tonic/latest/tonic/) for [CØSMOS SDK](https://docs.cosmos.network) based blockchains generated from their Protobuf definitions.

## Quick Start

```sh
cargo add cosmos-sdk-grpc-client
```
